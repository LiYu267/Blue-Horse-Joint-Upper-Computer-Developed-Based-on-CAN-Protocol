# Blue-Horse-Joint-Upper-Computer-Developed-Based-on-CAN-Protocol
基于CAN协议开发的蓝马关节上位机
电机监控上位机软件使用说明
软件概述
本上位机软件是基于C++/CLI开发的电机监控系统，通过CAN总线与电机驱动器通信，实现对电机状态的实时监控和精确控制。
系统要求
* 操作系统：Windows 11
* 运行环境：.NET Framework 3.8
* 硬件要求：USB-CAN适配器（支持A1+型号）
* 驱动：LM_USB驱动程序
功能特性
1. 设备连接管理
* 自动检测CAN设备
* 支持1MHz通信波特率
* 设备地址默认0x01
2. 状态监控
* 实时显示母线电压
* 实时显示母线电流
* 实时显示工作温度
* 自动读取模式（5秒间隔）
3. 电机控制
* 电流控制：Q轴电流控制
* 速度控制：转速控制
* 位置控制：绝对位置控制，单位Count（1圈=16384Count）
* 原点回归：最短路径回零功能
4. 故障诊断
* 实时故障码解析
* 故障类型详细说明
* 一键清除故障功能
* 颜色状态指示（绿色正常/红色故障）
快速开始
连接设备
1. 连接USB-CAN适配器到电脑
2. 连接CAN总线到电机驱动器
3. 点击"连接设备"按钮
4. 查看日志确认连接状态
基本操作
1. 电机使能：点击"电机使能"按钮激活电机
2. 状态监控：开启"自动读取"实时监控状态
3. 故障处理：出现故障时点击"清除故障"
4. 电机控制：在对应文本框中输入值然后再点击对应控制按钮
控制参数说明
电流控制
text
输入示例：1.5  (表示1.5A)
转换公式：实际值 = 输入值 × 1000
速度控制
text
输入示例：100  (表示100RPM)
转换公式：实际值 = 输入值 × 100
位置控制
text
输入示例：90   (表示90度)
转换公式：实际值 = 输入值 × 16384 / 360
故障代码说明
* 0x01：电压故障 - 母线电压异常
* 0x02：电流故障 - 母线电流异常
* 0x04：温度故障 - 电机或驱动器过热
* 0x08：编码器故障 - 编码器信号异常
* 0x40：硬件故障 - 驱动器硬件异常
* 0x80：软件故障 - 系统软件异常
注意事项
1. 操作前确保机械结构无干涉
2. 首次使用建议从低速、小电流开始测试
3. 出现故障时及时停止操作并排查原因
4. 定期检查连接线缆是否牢固
5. 避免在高温、潮湿环境下使用
技术支持
如遇问题请检查：
* CAN连接线是否接好
* 设备地址是否正确
* 电源电压是否正常
* 驱动器参数是否匹配
版本信息
